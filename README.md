# Agreed with Apollo

## How to use

```
npm start
```

* localhost:3000 front
* localhost:3001 graphql
* localhost:3010 agreed

```
front <=> graphql <=> agreed
```

See `scheam.graphql`, `agreed.js`, `graphql-server/resolvers.js`

## How to check graphql

* open localhost:3001/graphiql => GraphiQL
